# 🚀 SalesFlow CRM UI System

A SaaS-like CRM frontend simulation system built using HTML, CSS, and Vanilla JavaScript.

---

## 🎯 Objective
To build a realistic CRM frontend that simulates real product behavior using dynamic UI and fake CRUD.

---

## ⚡ Current Status
🟢 Phase 2 – UI Simulation Engine (In Progress)

---

## 🛠 Tech Stack
- HTML5
- CSS3 (Flexbox + Grid)
- Vanilla JavaScript

---

## 🚀 Upcoming Features
- Enquiry list + detail view
- Create enquiry modal
- Fake CRUD (JS + localStorage)
- Dynamic table rendering
- Navigation flow system

---

## 📌 Note
This project is being actively built as part of a structured roadmap towards a full-stack SaaS CRM system.
